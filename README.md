# Terraform Infrastructure Examples

This repository contains basic Terraform examples used for learning and supporting infrastructure provisioning in AWS and Azure environments.

The examples here focus on common infrastructure components and follow simple, readable patterns rather than complex or production-ready architectures.

### What this repository covers:
- Basic infrastructure provisioning using Terraform
- Use of variables and outputs
- Simple AWS and Azure resource examples
- Understanding Terraform workflow (init, plan, apply)

These examples are intended for learning, reference, and supporting infrastructure automation tasks.
